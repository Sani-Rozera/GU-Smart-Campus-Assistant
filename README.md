# 🎓 Geeta University AI Assistant

An AI-powered chatbot that answers Geeta University-related queries using official PDF documents through Retrieval-Augmented Generation (RAG).

## 🚀 Features
- AI-powered Q&A with Google Gemini
- PDF-based document retrieval
- Semantic search using ChromaDB
- Interactive Streamlit interface
- Accurate answers using LangChain

## 🛠 Tech Stack
Python • Streamlit • LangChain • Google Gemini • ChromaDB • Hugging Face Embeddings

## ⚙️ Setup

1. Install dependencies
```bash
pip install -r requirements.txt
```

2. Create a `.env` file
```env
GOOGLE_API_KEY=YOUR_API_KEY
```

3. Add your PDF documents to the `data` folder.

4. Run the application
```bash
streamlit run app.py
```
